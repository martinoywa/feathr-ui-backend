# Feathr UI Backend(API)
API Backend for Feathr UI using Fast API

# Set Up
1. Install dependencies
```bash
pip install -r requeirements.txt
```
2. Run App
```bash
fastapi dev src/api/main.py
```
3. Play around with Swagger UI
```
http://127.0.0.1:8000/docs
```
